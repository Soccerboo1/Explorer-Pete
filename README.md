# Project Title
Explorer Pete Adventure 

## Project Description
A video game menu

## Intended User or Player
Explorer Pete Adventure's players 

## User / Player Goal
The player should be able to intract with the menu to what they want with the game, aka start game, see extras, or quit. 

## Inspiration Interfaces
List the inspiration interfaces you used and briefly explain what you learned from them.
Into the Breach- Having the menu art vibe well with the title. 
Star Wars Knights of the Old Republic- Having an emphasis on the menu buttons. 

## Design Choices
- Layout:menu button on the left, art on the right, title on top center. 
- Color: Green, Black, orange, to go with a Red Foot tortoise. 
- Typography: Used Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif as my font-family code. 
- Visual hierarchy: The start button is the biggest button, then extras and last being quit the smallest one. 
- Feedback states: Links change color when hoverd over. 
- Responsive design: The image resize with smaller screens.
